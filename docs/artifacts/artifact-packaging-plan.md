# Artifact Packaging Plan

Production package for **What Coffee Demands: Volume One -- Hold the Earth**.

## Individual Downloads

Approved final images will eventually live in:

`public/downloads/artifacts/`

Use the filenames defined in `docs/artifacts/artifact-manifest.json`. Do not add placeholder artwork, draft generations, private-reader data, or manuscript text to the public download folder.

## Future ZIP Pack

After approved images exist, create:

`public/downloads/WDB_What_Coffee_Demands_Artifact_Pack.zip`

The ZIP should contain only approved public artifact images and a short public-facing usage note if needed.

## Future Public Gallery

Plan a future route:

`/archive/artifacts`

Do not build the gallery yet. The route should wait until approved artwork exists and the public presentation order is finalized.

## First Release Set

Recommended first release artifacts:

- Wooden Elephant
- Southampton-to-Massawa Passage Tickets
- Brass Compass
- Travel Trunk
- Leather-Bound Portfolio
- Whisky Bottle and Glass

This first set establishes the emotional object, the journey, private planning, and domestic tension without requiring the full collection to be complete.

## Source And Approval Handling

Keep unapproved source files, intermediate generations, layered files, notes, and rejected artwork in:

`private/artifact-source/`

That folder is ignored by git and should not be committed.
